CQL Interpreter

This project involves the design and implementation of a Domain-Specific Programming Language (DSL), named CQL (CSV Query Language), specifically tailored for querying and transforming data within CSV (Comma Separated Values) documents. Developed in Haskell, CQL aims to provide a flexible and intuitive way to interact with structured CSV data, drawing inspiration from established query paradigms such as Datalog for deductive databases and SQL for relational databases.

Runway Redeclaration Tool

This project is a Runway Redeclaration Tool developed as part of the Software Engineering Group (SEG) coursework. It provides a decision-support system for airports when an obstacle is present on a runway, allowing operators to quickly recalculate runway parameters and determine if safe operations can continue.
